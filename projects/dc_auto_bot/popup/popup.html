Helper 사용법

먼저 helper 서버를 실행하고, 외부에서 접속해야 하면 다른 PowerShell 창에서 Cloudflare Tunnel을 실행합니다.

1. helper 실행
2. Cloudflare Tunnel 실행
터널이 만든 `https://xxxxx.trycloudflare.com` 주소로 바로 들어가지 말고, 뒤에 `/transparency`를 붙여서 접속하세요. 예: `https://xxxxx.trycloudflare.com/transparency`
3. helper 상태 확인
4. Windows에서 여전히 Gemini CLI를 못 찾을 때
메모
Gemini CLI가 설치되어 있고 로그인된 상태여야 판정이 동작합니다. helper를 끄려면 실행 중인 PowerShell 창에서 Ctrl+C를 누르면 됩니다. `Gemini CLI 실행 파일을 찾지 못했습니다`가 뜨면 PowerShell에서 `gemini --version`이 되는지 먼저 확인하세요. Gemini CLI를 방금 설치했다면 helper를 완전히 껐다가 다시 켜서 새 PATH를 읽게 해야 합니다. `Cannot use both a positional prompt and the --prompt (-p) flag together`가 떴다면 helper를 껐다가 다시 켜서 새 `stdin` 모드가 반영되게 하세요.
